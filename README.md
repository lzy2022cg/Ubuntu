/*Ubuntu镜像源更换/
由于直接从网站上下载的镜像源是国外的，国内的很多软件会无法下载，所以要更换镜像源，以下是操作步骤
阿里云镜像源网址https://developer.aliyun.com/mirror/
清华大学镜像源https://mirrors.tuna.tsinghua.edu.cn/
打开终端输入指令:sudo gedit sources.list
镜像源:
